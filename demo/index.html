Postman Button Demo

An embeddable widget for launching Postman Agent Mode or forking collections from any webpage.

Agent Mode Button

Injects a pre-written prompt into Postman's Agent Mode. The user lands in Postman with the prompt ready to send.

Embed code:

<!-- Agent Mode button -->
<div class="postman-button"
     data-postman-action="agent/prompt"
     data-postman-agent-prompt="Build a REST API for a todo app with CRUD endpoints"
     data-postman-utm-campaign="my-tutorial">
</div>

Agent Mode — Light Theme, Custom Label

Use data-postman-button-theme="light" and data-postman-button-label to customize appearance.

<div class="postman-button"
     data-postman-action="agent/prompt"
     data-postman-agent-prompt="Test the Stripe Payments API..."
     data-postman-auto-send="true"
     data-postman-button-label="Try with Postman AI"
     data-postman-button-theme="light"
     data-postman-utm-campaign="stripe-tutorial">
</div>

Fork Collection Button

Forks a public collection into the user's Postman workspace.

<!-- Fork Collection button -->
<div class="postman-button"
     data-postman-action="collection/fork"
     data-postman-collection-id="12959542-c8142d51-e97c-46b6-bd77-52bb66712c9a"
     data-postman-utm-campaign="api-docs">
</div>

Multiple Buttons

Place as many buttons as you want on a single page. Each reads its own data-* attributes independently.

Configuration Reference

Attribute Required Default Description
data-postman-action Yes — agent/prompt or collection/fork
data-postman-agent-prompt Agent mode — Plain text prompt (auto Base64-encoded)
data-postman-auto-send No false Auto-send the prompt on load
data-postman-collection-id Fork mode — Public collection UID to fork
data-postman-utm-campaign No postman-button UTM campaign name for analytics
data-postman-utm-content No Auto (full URL) Set to "false" to disable page URL tracking
data-postman-button-label No Auto Override the button label text
data-postman-button-theme No dark dark (orange) or light (white)
